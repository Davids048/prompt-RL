Nuances recorded
----------------
1. Text-to-video nuance: The paper says VideoAlign evaluates visual aesthetics quality, motion quality, and text-video
alignment. However, it also says the text-video alignment dimension was unstable and was excluded from the final
analysis. Table 5 still reports TA and VisionReward-Video alongside VQ and MQ.

2. Image-to-video nuance: The paper says I2V models exclusively allow optimization of motion quality because visual
fidelity and text-video alignment are constrained by the input image rather than the model parameter space.

3. Prompt optimization nuance: I did not find prompt rewriting, prompt enhancement, or learned prompt optimization. The
paper discusses using prompt datasets to optimize the models, not optimizing prompts themselves.

4. T2I prompt source nuance: The paper says the T2I optimization prompt set is curated to balance diversity and
complexity, but it does not name the source dataset for that curated T2I prompt set.
